# Skills — 项目级技能模块

> 此目录存放项目特定的 UI 技能定义（如 frontend-design, ui-ux-pro-max）
> 
> Claude Code 内置 Skills（brainstorming, writing-plans, TDD, systematic-debugging,
> using-git-worktrees, verification-before-completion, requesting-code-review, simplify）
> 通过 Skill 工具按场景触发，无需在此定义。
>
> 此目录用于存放项目自定义的额外技能规范。

## 内置 Skills 触发场景

| Skill | 触发条件 |
|-------|---------|
| brainstorming | 新功能/新需求 |
| writing-plans | 多步骤任务 |
| test-driven-development | 写代码 |
| systematic-debugging | 遇到 bug |
| using-git-worktrees | 开始开发 |
| verification-before-completion | 即将完成 |
| requesting-code-review | 代码写完 |
| simplify | 审查阶段 |

## 前端专属 Skills

| Skill | 触发条件 |
|-------|---------|
| frontend-design | UI 改动 |
| ui-ux-pro-max | UI 改动 |
