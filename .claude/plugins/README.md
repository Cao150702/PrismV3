# Plugins — 扩展能力插件

> 此目录存放扩展 Claude Code 能力的插件定义
> 例如：Context7 集成、Exa 搜索集成、mem0 记忆集成等

## 建议配置的插件

### Context7（实时文档查询）
- 用途：查询第三方库的最新 API 文档
- 触发：研究路由规则中"查 API 用法"时

### Exa（AI 语义搜索）
- 用途：搜索最新技术最佳实践
- 触发：研究路由规则中"查最佳实践"时

### mem0（云端记忆）
- 用途：跨项目的长期记忆存储
- 触发：记录用户偏好、产品愿景、长期反馈

### Playwright（浏览器自动化）
- 用途：E2E 测试
- 触发：QA Engineer 执行前端验收测试时
